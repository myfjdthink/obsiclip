# Change: 添加多语言支持 (i18n)

## Why

ObsiClip 目前界面文本全部硬编码为中文，限制了非中文用户的使用体验。为了扩大用户群体并提供更好的本地化体验，需要实现多语言支持。首次安装时应根据浏览器语言自动选择合适的语言，同时允许用户手动切换。

## What Changes

本次变更实现完整的国际化 (i18n) 支持：

- **i18n**: 新增国际化能力
  - 创建语言资源文件（中文 zh-CN、英文 en）
  - 实现语言检测与自动适配逻辑
  - 提供语言切换功能
  - 更新所有 UI 组件使用翻译函数

## Impact

- 受影响的规格: 新增 `i18n` 能力规格
- 受影响的代码:
  - `utils/i18n.ts` - 新增 i18n 工具函数
  - `locales/` - 新增语言资源目录
  - `entrypoints/popup/**` - 更新所有 Popup 组件
  - `entrypoints/settings/**` - 更新设置页面组件
  - `utils/storage.ts` - 新增语言偏好存储
- 技术方案: 轻量级自定义 i18n 方案，无需引入额外依赖
